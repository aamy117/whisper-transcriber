使用說明

快捷鍵

  • Space - 播放/暫停
  • Ctrl + ← - 快退 3 秒
  • Ctrl + → - 快進 3 秒
  • Ctrl + ↑ - 加速播放
  • Ctrl + ↓ - 減速播放
  • Ctrl + T - 插入時間戳
  • Ctrl + S - 儲存編輯
  • Ctrl + F - 搜尋文字
  • Ctrl + E - 匯出檔案

使用步驟

  1. 設定您的 OpenAI API Key（點擊右上角設定按鈕）
  2. 上傳音訊檔案（支援 MP3、WAV 等格式，最大 50MB）
  3. 點擊「開始轉譯」按鈕
  4. 等待轉譯完成後，即可編輯文字內容
  5. 點擊段落可跳轉到對應時間點播放
  6. 編輯完成後可匯出為多種格式
