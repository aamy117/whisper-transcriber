🚨 需要啟動本地伺服器

❌ CORS 錯誤

你遇到了這個錯誤是因為直接從檔案系統開啟 HTML（使用 file:// 協議）。

瀏覽器的安全政策不允許從 file:// 載入 JavaScript 模組。

🔧 解決方法：啟動本地伺服器

方法 1：使用 Python（推薦）

如果你已安裝 Python，在專案目錄執行：

python -m http.server 8000

或 Python 2：

python -m SimpleHTTPServer 8000

方法 2：使用批次檔（Windows）

雙擊執行專案中的 start-server.bat

方法 3：使用 VS Code Live Server

  1. 安裝 VS Code
  2. 安裝 "Live Server" 擴充功能
  3. 在 video.html 上按右鍵，選擇 "Open with Live Server"

方法 4：使用 Node.js

npm install -g http-server
http-server -p 8000

✅ 啟動伺服器後

在瀏覽器訪問：

http://localhost:8000/video.html


前往視訊播放器 瀏覽所有檔案
